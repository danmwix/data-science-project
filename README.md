# data-science-project
Global Video Game Sales Analysis 🎮This project analyzes global video game sales trends using Python, NumPy, Pandas, and Matplotlib. It satisfies the requirements for the Applied Sciences course at Thomas More.📁 Repository Structuredata/: Contains the original and cleaned datasets.scripts/: Python scripts for data cleaning using Regex.notebooks/: Jupyter Notebooks for NumPy and Pandas analysis.README.md: Setup instructions and project documentation.🚀 Setup Instructions for Team MembersFollow these steps exactly to ensure the project runs on your machine.1. Clone the RepositoryOpen your terminal (or VS Code) and run:PowerShellgit clone https://github.com/danmwix/data-science-project.git
cd data-science-project
2. Create the Virtual EnvironmentWe are using Python 3.13.2. Create a virtual environment to keep dependencies isolated:PowerShellpython -m venv venv
3. Activate the EnvironmentWindows (PowerShell): .\venv\Scripts\Activate.ps1Mac/Linux: source venv/bin/activateNote: You should see (venv) at the start of your command line.4. Install DependenciesRun this command to install all necessary libraries:PowerShellpip install pandas numpy matplotlib seaborn notebook
🛠 Fixing "Import Could Not Be Resolved" (VS Code)If you see yellow squiggles under import pandas or import numpy, VS Code is not using the correct interpreter.Open clean_data.py or one of the notebooks.Press Ctrl + Shift + P and type "Python: Select Interpreter".Choose the one labeled Python 3.13.x ('venv': venv).For Notebooks: Click "Select Kernel" in the top right corner and choose the same venv path.🔍 How to Run the ProjectPhase 1 (Data Cleaning): Run the Regex cleaning script from the terminal:PowerShellpython scripts/clean_data.py
This will generate data/vgsales_cleaned.csv.Phase 2 & 3 (Analysis):Open the files in the notebooks/ folder and click "Run All".⚠️ Challenges & TroubleshootingChallengeCauseFixKeyboardInterrupt / HangingOneDrive syncing while creating venv.Delete venv folder and run python -m venv venv again; wait 60s.ImportError (Matplotlib)Corrupted install or interrupted pip.Run pip install --upgrade --force-reinstall matplotlib.FileNotFoundErrorNotebook looking in the wrong directory.Ensure the path is ../data/vgsales_cleaned.csv if running inside notebooks/.Regex IssuesYear column contained floats (2008.0).We used re.search(r'(\d{4})') to extract only the 4-digit integer.🎓 Project Highlights (For Presentation)Regex: Used to clean non-ASCII characters from game titles (like "Pokémon") and to group console platforms into "Sony", "Nintendo", and "Microsoft" brands.NumPy: Leveraged for statistical math (Mean, Std Dev) on 16,000+ rows.Pandas: Used Pivot Tables and Heatmaps to prove that specific publishers dominate certain genres (e.g., Electronic Arts owns the Sports genre).
